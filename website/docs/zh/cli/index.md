# CLI 参考

pyscn 提供四个顶级命令：

| 命令 | 用途 |
| ------- | ------- |
| [`analyze`](analyze.md) | 运行所有分析并生成报告（默认为 HTML）。 |
| [`check`](check.md)     | 用于 CI/CD 的快速、严格质量门禁。退出码 0/1/2。 |
| [`init`](init.md)       | 生成带注释的 `.pyscn.toml` 配置文件。 |
| [`version`](version.md) | 输出版本信息。 |

全局选项 `-v / --verbose` 适用于所有命令。
