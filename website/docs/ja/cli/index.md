# CLI リファレンス

pyscn は 4 つのトップレベルコマンドを提供します:

| コマンド | 用途 |
| ------- | ------- |
| [`analyze`](analyze.md) | すべての解析を実行し、レポートを生成します（デフォルトは HTML）。 |
| [`check`](check.md)     | CI/CD 向けの高速で厳格な品質ゲートです。終了コード 0/1/2。 |
| [`init`](init.md)       | コメント付きの `.pyscn.toml` 設定ファイルを生成します。 |
| [`version`](version.md) | バージョン情報を表示します。 |

グローバルフラグ `-v / --verbose` はすべてのコマンドで使用できます。
