# 設定

pyscn は `.pyscn.toml` または `pyproject.toml` 内の `[tool.pyscn]` から設定を読み込みます。CLI フラグは設定ファイルより優先され、設定ファイルは組み込みデフォルトより優先されます。

- **[設定ファイル形式](format.md)** — 探索ルール、ファイルの場所、優先順位。
- **[リファレンス](reference.md)** — すべての設定キー、型、デフォルト値、効果。
- **[設定例](examples.md)** — 厳格な CI、大規模コードベース、最小限のオーバーライド。

コメント付きのスターターファイルを生成するには:

```bash
pyscn init
```
